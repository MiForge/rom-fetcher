# ROM Fetcher

A simple browser-based tool to find and download Xiaomi firmware by device codename.

**Live:** https://miforge.github.io/rom-fetcher

No backend. All firmware data is fetched directly from [XiaomiFirmwareUpdater/miui-updates-tracker](https://github.com/XiaomiFirmwareUpdater/miui-updates-tracker). No files are hosted or redistributed here.
